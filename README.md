## Что такое SVM?
# SVM (Support Vector Machine) - это алгоритм машинного обучения для классификации и регрессии. Его основная идея - найти оптимальную границу между классами, которая максимизирует запас (margin) - расстояние до ближайших точек каждого класса.

**Ключевые концепции:**
Опорные векторы - точки данных, ближайшие к разделяющей поверхности

Гиперплоскость - разделяющая поверхность в n-мерном пространстве

Запас (Margin) - расстояние от гиперплоскости до ближайших точек

Ядерный трюк - работа в высокоразмерных пространствах без явных вычислений



# Ключевые преимущества SVM:
Эффективность в высокоразмерных пространствах

Работа с нелинейными данными через ядра

Устойчивость к переобучению (хорошая обобщающая способность)

Оптимальная разделяющая поверхность

*Когда использовать SVM:*
✅ Средние размеры datasets

✅ Высокоразмерные данные

✅ Четкие границы между классами

✅ Нелинейные зависимости

❌ Очень большие datasets

❌ Зашумленные данные с перекрывающимися классами

❌ Требуется вероятностная интерпретация
